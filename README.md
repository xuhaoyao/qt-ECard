**功能需求：**

实现一个制作贺卡的小软件。具体功能需求如下：

（1）        用户可在软件中选择背景:要求能列出可选的背景图片

（2）        用户可在软件中输入标题

（3）        用户可在软件中输入贺卡中的文字

（4）        用户可在软件中选择贺卡的伴奏音乐

（5）        用户可保存贺卡（即保存后关闭程度后，可以重新打开、播放）

在这里的每一个步骤，都可以回到上一步，并且再设置向前后步骤走时，原来的设置不会被清除

**相关技术：**

（1）        文字、图像展现

（2）        音乐播放

（3）        控件使用
